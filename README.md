販売管理

## 販売管理とは
```
自社の商品とお金がどのように動いているのかを管理することが販売管理
```
**※ 請求・入金**

## 管理すべき情報
- 見積もり
- 受注
- 出荷
- 納品( 受領書 )
- 売上
- 請求
- 入金
- 
- 発注
- 入荷( 検品 )
- 仕入
- 支払い
-
- 在庫管理
- 在庫棚卸し

## エンドユーザー
**業務フローの目的**

## 処理のサイクル
- 日次処理
- 月次処理
- 随時処理
